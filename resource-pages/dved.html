CSS Grid Tutorial

Takeaways

  • Use repeat() as a shorter syntax for defining columns and rows.
  • Row and column lines are counted, starting at 1.
  • If you use grid-template-areas, you don't need to count lines.
  • Grid-template-areas is a visual representation of your grid in code.
            
    main {
        display: grid;
        grid-template-columns: repeat(2, 1fr);
        grid-template-rows: repeat(2, 1fr);
        grid-template-areas:
            "box1 box2"
            "box3 box3";
    }
            
            
            
    .box1 {
        grid-column: 1/2;
        grid-row: 1/2; 
    }

    /* or, use grid-areas */
    
    .box1 {
        grid-area: box1;
    }
