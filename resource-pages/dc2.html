Responsive Grid - No Media Queries

Takeaways

  • Autofit and autofill both automatically adjust columns in CSS Grid. These allow for responsiveness without media queries.
  • These keywords manage column sizing and wrapping, according to the content and space available.
  • Autofit fits the current columns in the space available.
  • Autofill expands and adds more columns to take up any extra space in the grid container.
            
    main {
        display: grid;
        grid-template-columns: repeat(auto-fit, minmax(20rem, 1fr));
        grid-gap: 1em; 
    }
