Grid-Template-Areas

Takeaways

  • Structure your html according to how the elements will fit in your grid.
  • You can use grid-template-areas with media queries to make your grid responsive.
  • To do this, copy and paste grids for different screen sizes, adjust columns and rows, and move grid-areas accordingly.
  • This thinking can be applied to nested grids as well.
            
    body {
        display: grid;
        grid-template-columns: 100%;
        grid-template-rows: repeat(4, 1fr);
        grid-template-areas:
            "nav"
            "sect1"
            "sect2"
            "footer";
    }
            
            
            
    @media only screen and (min-width: 768px) {
        body {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: auto 2fr 10%;
            grid-template-areas:
                "sidebar nav nav"
                "sidebar sect1 sect2"
                "footer footer footer";
        }
    }
